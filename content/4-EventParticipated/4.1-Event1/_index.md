### Non-Determinism of "Deterministic" LLM Settings
#### Key Takeaways

- Non-determinism tied to GPU architecture, floating-point accuracy &
inference optimization
- temp=0 remove sampling randomness, but not all sources of non-determinism
Try set temp=0.1 if encounter response loops
Remember to Read the Docs
- Build with variance in mind
- Multiple runs & Ensembles (cost–accuracy tradeoffs)
- Enforce structured outputs with params, not just prompts
- Design to handle probabilistic
