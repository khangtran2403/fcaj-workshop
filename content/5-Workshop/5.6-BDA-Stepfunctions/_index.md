#### Amazon Bedrock Data Automation and AWS Step Functions

In this lab, you will create a Blueprint in Amazon Bedrock Data Automation (BDA) to extract key fields that will be passed to an AWS Lambda function. You will then build a workflow using AWS Step Functions and Amazon EventBridge.
