#### Build the Storage and Database Layer

In this section, you will create three Amazon S3 buckets: one for storing input documents, one for storing the output generated after processing by Amazon Bedrock Data Automation (BDA), and one for storing static assets to be deployed through Amazon CloudFront. You will also create an Amazon DynamoDB table to store the normalized data.
